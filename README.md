# FuelEU Maritime Compliance Platform

Monorepo scaffold for a FuelEU Maritime compliance platform.

## Structure

- frontend/ → React + TypeScript + Vite + TailwindCSS
- backend/ → Node.js + TypeScript + Express + PostgreSQL
- AGENT_WORKFLOW.md
- REFLECTION.md

## Current Status

This stage includes project structure, strict TypeScript configuration, and boilerplate only.
No business logic has been implemented yet.

## Run

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm run dev
```
